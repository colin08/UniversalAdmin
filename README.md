# UniversalAdmin
通用版后台管理框架

----------

搭建此项目是为了给自己做个积累吧，学习新的东西就集成进来，方便以后直接使用

开发工具:VS2015

开发框架:MVC5,EF6.1

主要功能模块：

1. 后台用户模块
2. 无限级分类
3. 权限管理
4. 系统日志
5. 数据库备份
6. 站点配置文件
7. 集成WebApi
8. 使用SignalR搭配SqlDependency来做站内及时消息提醒
8. Demo模块来演示相册的处理和强类型的动态表单
